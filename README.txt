# ForgeHer PWA

This package is an installable web app for iPhone Home Screen use.

## How to use
1. Upload these files to any simple static host:
   - GitHub Pages
   - Netlify
   - Vercel
   - Cloudflare Pages
2. Open the hosted URL in Safari on iPhone.
3. Tap Share > Add to Home Screen.

## Notes
- The looping demos are embedded locally as animated SVG files, so they stay inline.
- This is a PWA, not a native App Store build.
